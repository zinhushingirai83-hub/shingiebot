Elite AI Football Betting Analyst — Setup Guide
✅ All your data (API key, slips, preferences) is stored ONLY on your device. Nothing is sent to any server except directly to Anthropic's API and TheSportsDB for fixtures.

📱 Install as PWA on Samsung M33

1
Open in Chrome
Make sure you're using Google Chrome on your Samsung M33. Open the ShingieBot URL in Chrome.
2
Add to Home Screen
Tap the three dots menu (⋮) in Chrome → tap "Add to Home screen" → tap "Add". ShingieBot will appear on your home screen like an app.
3
Get Your Anthropic API Key
Go to console.anthropic.com → sign up or log in → click "API Keys" → create a new key. Copy it.
sk-ant-api03-...
You need to add credit to your Anthropic account. Analysis costs roughly $0.01–$0.03 per slip generation.
4
Enter API Key in ShingieBot
Open ShingieBot → tap the "API" button in the top right → paste your key → tap "SAVE KEY". Analysis will begin immediately.
5
Daily Routine
Open the app every morning. Your 1.5 SAFE slip generates automatically for today's matches. Check the FIRE tab for recovery. On Mondays, the WEEKLY slip generates for the full week.
